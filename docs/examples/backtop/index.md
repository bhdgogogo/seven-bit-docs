## API

### `<back-top>` Props

| 参数名     | 描述                                 | 类型   | 默认值 |
| :--------- | :----------------------------------- | :----- | :----- |
| right(px)  | 返回顶部按钮出现的位置距离最右侧距离 | Number | 40     |
| bottom(px) | 返回顶部按钮出现的位置距离最底部距离 | Number | 40     |
| height(px) | 返回顶部按钮出现的位置               | Number | 200    |

### `<back-top>` Slots

| 插槽名  | 描述               | 参数 |
| :------ | :----------------- | :--- |
| default | 返回顶部容器的内容 | -    |
