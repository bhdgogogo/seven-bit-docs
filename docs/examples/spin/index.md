## API
### `<spin>` Props

| 参数名  | 描述                                   | 类型      | 默认值  |
| :------ | :------------------------------------- | :-------- | :------ |
| size    | 尺寸                                   | `number`  | `20`     |
| loading | 是否为加载中状态（仅在容器模式下生效） | `boolean` | `false` |
| dot     | 是否使用点类型的动画                   | `boolean` | `false` |
| tip     | 提示内容                               | `string`  | `-`     |

### `<spin>` Slots

| 插槽名  | 描述                   | 参数 |
| :------ | :--------------------- | :--- |
| default | 放置需要展示加载中的容器             | -    |
| icon    | 自定义图标（自动旋转） | -    |
